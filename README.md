# Silence
**Console Chat App Built With AssemblyScript**

## Features
- DM Messaging in the console
- Private 1-1 Chatting
- Uses AssemblyScript WebSocket
- Implements Console bindings
- Start with a CLI command

## Installation

```bash
~ npm i as-silence -g
```

## Usage

```bash
~ silence
```
